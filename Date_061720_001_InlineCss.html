NOTE: With internal CSS disadvantage is that, if we want to remove all styling from a web page, we can simply do that by removing all code from within the style tag in the head section. But if we had used inline css, then it becomes extremely difficult to remove/diffuse the styling of all elements because then styling at each individual element should be removed.
